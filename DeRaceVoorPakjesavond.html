De Race Voor Pakjesavond

De race voor pakjes avond is a racing game at night set in Amsterdam and Madrid. You race as Sinterklaas in a singleplayer or local splitscreen 1v1 through the streets of Amsterdam and Madrid. Your goal is to get the fastest lap time on a map.


The project me made was a racing game set at night in 4 weeks. In addition we did have a group of 3 working on the project. Our idea came from it almost being the time of the year where Sinterklaas would be celebrated in the Netherlands so we thought it a funny idea to theme it in this holiday.



In this snippet the OnMove() is the function that gets the player input from the new input and put it in a vector2. The MovePlayer() function is in charge to move the player for both players in one system.




This was the first time using the Unity playerprefs and I wanted to make a system so the player can change his/her controls for the player and it to be saved for a next time. I make a quite efficient system to get the input from the input field in unity and transfer it to the control of the car and save it aswell in the playerprefs so the controll was saved.
